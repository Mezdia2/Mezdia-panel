# Changelog

## 3.2.0
- بازطراحی کامل رابط کاربری پنل (پوسته تاریک/روشن، تایپوگرافی، ساختار网格)
- جایگزینی فونت‌های SF Pro با Vazirmatn و JetBrains Mono
- طراحی مجدد هدر ناوبری با برچسب‌های انگلیسی و جداکننده گروهی
- بهبود نمای QR کد با فریم اسکن
- بازطراحی صفحه‌های نمای کلی، حساب، تنظیمات، API و گزارش‌ها
- هماهنگ‌سازی نسخه‌ها: worker.js، worker-template.txt و سیستم به‌روزرسانی خودکار
- رفع عدم تطابق نسخه پنل و سیستم به‌روزرسانی (PANEL_VERSION)
